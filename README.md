## Informations, install and cogs.
`[p]` is your prefix.

To add this repo:

```yaml
[p]repo add frcogs https://github.com/Mrs-Feathers/frcogs
```

To install any cog you want:

```yaml
[p]cog install frcogs <cog name you want>
```
